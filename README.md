# 🛡️ Chatbot for Identifying Phishing Emails, Messages, and Websites

A machine learning-powered chatbot designed to help users detect phishing content across emails, messages (SMS), and websites. This tool leverages natural language processing (NLP) techniques to classify suspicious content and raise user awareness about potential security threats.

---

## 🔍 Features

- 🔐 Detect phishing attempts in:
  - Emails
  - Text messages
  - URLs/websites
- 🤖 Interactive chatbot interface
- 🧠 ML/NLP-based classification model
- 📊 Real-time threat score feedback
- 🛠️ Modular and extendable design
